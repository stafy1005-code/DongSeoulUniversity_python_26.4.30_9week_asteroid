# 🚀 3D Asteroid Shooting Game

## 📌 프로젝트 소개

이 프로젝트는 Python과 Pygame을 사용하여 제작한 우주 슈팅 게임이다.  
기존 Asteroid 게임의 기본 구조를 바탕으로, 평면적인 탑다운 방식이 아니라 플레이어 뒤에서 전방을 바라보는 **3인칭 후방 원근 시점**을 적용하였다.

플레이어는 우주선을 조종하여 전방에서 다가오는 소행성, 외계인, 적 탄환을 피하거나 파괴하며 진행한다.  
일반 소행성 슈팅 구조에 아이템, 랭크업, 보스전, 자석 효과, 미사일, 저격 모드 등을 추가하여 게임성을 확장하였다.

3. 삼각함수 기반 이동 알고리즘

총알, 미사일, 적 탄환은 목표 방향으로 이동해야 한다.
이를 위해 두 점 사이의 x, y 차이를 구한 뒤, 거리로 나누어 방향 벡터를 만든다.

dx = target_x - start_x
dy = target_y - start_y
distance = math.hypot(dx, dy)

move_x = dx / distance * speed
move_y = dy / distance * speed

이 알고리즘을 사용하면 오브젝트가 목표 지점을 향해 일정한 속도로 이동할 수 있다.

4. pseudo-3D 원근 표현

Pygame은 기본적으로 2D 게임 라이브러리이지만, 이 게임에서는 z 값을 추가하여 3D처럼 보이게 만들었다.

오브젝트는 실제로는 2D 화면에 그려지지만, 내부적으로는 다음과 같은 좌표를 가진다.

x = 좌우 위치
y = 상하 위치
z = 깊이 / 거리

z 값이 클수록 멀리 있는 오브젝트이고, z 값이 작아질수록 플레이어에게 가까워진다.

화면에 그릴 때는 z 값을 이용해 크기와 위치를 조절한다.

scale = PROJECTION / z
screen_x = CENTER_X + x * scale
screen_y = CENTER_Y + y * scale
size = base_size * scale

이 방식으로 소행성이나 아이템이 멀리 있을 때는 작게 보이고, 가까워질수록 크게 보이게 만들었다.

5. 별 배경 원근 효과

배경의 별도 단순한 고정 이미지가 아니라, 여러 개의 작은 점 오브젝트로 생성된다.
별들은 각각 다른 위치와 깊이값을 가지고 있으며, 플레이어가 움직이면 별들이 반대 방향으로 움직이는 것처럼 보인다.

이를 통해 플레이어가 실제 우주 공간 안에서 움직이는 듯한 입체감을 만든다.

일반 별은 소행성과 헷갈리지 않도록 다음과 같이 표현했다.

아주 작은 점
흰색, 푸른색, 노란빛 계열
약한 반짝임 효과
다각형이 아닌 원형 점 형태
6. 가속 / 감속 물리

플레이어 우주선은 키를 누른 즉시 고정 속도로 움직이는 방식이 아니라, 가속도와 감속을 사용한다.

키를 누르면 속도가 증가하고, 키를 떼면 마찰값에 의해 서서히 줄어든다.

velocity_x += acceleration_x
velocity_y += acceleration_y
velocity_z += acceleration_z

velocity_x *= friction
velocity_y *= friction
velocity_z *= friction

이 방식으로 우주선이 실제로 미끄러지듯 움직이는 느낌을 줄 수 있다.

9. 랭크업 자석 효과

기존 레벨업 보상 기능을 랭크업 기능으로 분리하였다.
소행성이나 적을 처치하여 조건을 만족하면 랭크가 상승하고, 랭크업 시 10초 동안 자석 효과가 발동한다.

자석 효과가 발동하면 주변 아이템이 플레이어 쪽으로 빨려 들어온다.

dx = player_x - item_x
dy = player_y - item_y
distance = math.hypot(dx, dy)

item_x += dx / distance * magnet_speed
item_y += dy / distance * magnet_speed

이 기능은 랭크업 보상 느낌을 주고, 플레이어가 아이템을 놓치지 않도록 도와준다.

10. 보스전 알고리즘

보스는 처음에는 큰 발광 구체 형태로 등장한다.
플레이어가 목표 지점에 가까워지면 색이 점점 붉어지고, 이후 거대한 외계인 형태로 변신한다.

보스 특징:

일반 별보다 훨씬 큰 발광 구체
빛 번짐과 코로나 효과
가까워질수록 붉은색으로 변화
변신 후 거대 외계인 형태로 전환
작은 붉은 공과 큰 붉은 공을 발사
보스 처치 시 레벨 상승

보스는 일반 좌클릭 탄환으로는 피해를 입지 않는다.
보스를 공격하려면 마우스 오른쪽 버튼을 누른 상태에서 왼쪽 버튼을 눌러 저격탄을 발사해야 한다.

이렇게 한 이유는 보스의 피격 범위가 너무 넓어 일반 소행성이나 적 탄환을 대신 맞아버리는 문제를 줄이기 위해서이다.

11. 보스 저격 모드

보스전에서는 일반 탄환과 보스 공격 탄환을 구분한다.

일반 좌클릭 탄환: 소행성, 외계인, 붉은 공 처리
우클릭 유지 + 좌클릭 탄환: 보스 약점 공격

보스는 중심부의 작은 약점 판정만 피해를 받는다.
따라서 플레이어는 보스를 공격할 때 의도적으로 저격 모드를 사용해야 한다.

적용 효과:
